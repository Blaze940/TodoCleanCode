# Running the app 🏃
- **Adding todo** : python main.py "add" "New todos"
- **Removing todo** : python main.py "remove" "123" 

# Description 📝
The project is a python app, the main features are :
- Add a new todo.
- List all todos.
- Mark a todo as done.
- Remove a todo.
- Generate a report of completed tasks as txt.
